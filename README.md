# Guitar Tuner

A modern, real-time Guitar Tuner app built natively for Android. Designed with a clean, stock-like, and bloat-free user interface, this app helps you tune your guitar accurately using the device's microphone.

## ✨ Features
* **Real-time Pitch Detection:** Utilizes the YIN algorithm for highly accurate frequency tracking.
* **Multiple Tuning Modes:** Supports Standard, Drop D, Half-Step Down, Open G, Open D, and DADGAD.
* **Modern UI:** Built entirely with Jetpack Compose featuring a responsive, clean, and minimalist design.
* **Visual Tuning Needle:** Smooth, state-driven needle animations indicating Sharp, Flat, or In-Tune states.

## 🛠 Tech Stack & Architecture
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Architecture:** MVVM (Model-View-ViewModel)
* **Asynchronous Programming:** Coroutines & StateFlow
* **Audio Processing:** [TarsosDSP](https://github.com/JorenSix/TarsosDSP)
